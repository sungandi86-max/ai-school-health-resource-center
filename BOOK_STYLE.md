# BOOK_STYLE

## 문서 목적

이 문서는 『보건교사를 위한 AI 업무 자동화』 최종 조판본에서 확인되는 책 전체의 편집·시각 스타일을 기록합니다.

새로운 디자인 시스템을 제안하는 문서가 아닙니다. 아래 내용은 최종 PDF의 페이지 구성, 글꼴 정보, 색상값, 반복 컴포넌트와 실제 배치를 확인해 정리한 것입니다.

## 확인 범위

- 최종 조판본 전체 211쪽
- 표지와 차례
- 8개 PART 표지
- 22개 Chapter 표지와 본문
- 프로젝트 진행 카드
- CASE, Workflow, 실습, 표, 안내 상자
- 21개 자료실 QR 배치 페이지
- PART 마무리와 에필로그
- PDF 내부 글꼴·페이지·이미지 정보

## 판형과 기본 인상

### 확인된 규칙

- 최종 PDF의 페이지 크기는 419.528 × 595.276pt이다.
- 이 크기는 세로형 A5 판형에 해당한다.
- 전체 211쪽이 같은 세로 판형을 사용한다.
- 기본 본문 페이지는 흰색 바탕이다.
- 표지와 Chapter 표지는 짙은 네이비 전면 배경을 사용한다.
- 장식보다 제목, 선, 표, 상자와 여백으로 위계를 만든다.
- 대부분의 페이지는 한 단 구조다.
- 본문 기준선은 대체로 좌우 약 45.4pt 지점에서 시작하고 끝난다. 페이지 폭을 기준으로 약 16mm 안쪽이다.
- 카드와 안내 상자는 본문 기준선과 같거나 그보다 한 단계 안쪽에 배치된다.

### 색상 사용 방식

- 네이비는 책의 기본 식별색이자 강한 구조 구분에 사용된다.
- 그린은 진행, PART 구분, 작업 구조를 표시하는 강조색으로 사용된다.
- 골드와 연한 크림은 본문 흐름에서 별도로 읽어야 하는 해석·주의 성격의 상자에 사용된다.
- 청회색과 회색은 Preview, 테두리, 보조 정보처럼 위계를 낮추는 데 사용된다.
- 본문 문단에 여러 강조색을 섞지 않는다. 색은 주로 페이지 유형과 컴포넌트를 구분한다.

## 글꼴

### PDF에서 확인된 글꼴

- NotoSansKRThin-Regular
- NotoSansKRThin-Bold
- NotoSansKR-Regular
- DejaVuSans
- Helvetica

실제 한글 본문과 제목에서는 **Noto Sans KR Thin 계열의 Regular와 Bold가 중심**이다. NotoSansKR-Regular와 DejaVuSans는 체크 표시와 기호 등 일부 요소에서 함께 확인된다.

### 확인된 크기 위계

| 용도 | 확인된 크기 |
|---|---:|
| 표지 책 제목 | 29pt |
| PART 표지 제목 | 24pt |
| Chapter 제목 | 23pt |
| Chapter 표지의 큰 번호 | 62pt |
| 본문 안 주요 소제목 | 16pt |
| 본문 | 10.6pt |
| 표·카드의 보조 본문 | 9.5pt |
| 작은 라벨·표 내부 텍스트 | 7.5–7.8pt |
| 페이지 하단 정보 | 8pt |

### 글자 위계의 표현

- 제목 위계는 크기, 굵기, 색상과 페이지 분리로 만든다.
- Chapter 제목은 별도의 네이비 표지 페이지에 흰색으로 배치한다.
- 본문 안 소제목은 본문보다 크고 굵게 표시한다.
- 긴 본문은 검정색 Regular를 사용한다.
- 카드명, 표 머리글, CASE·Workflow·실습 제목은 굵기와 상자 또는 선으로 본문과 분리한다.
- 밑줄, 그림자, 입체 효과는 확인되지 않는다.

## 페이지 유형

### 1. 책 표지

- 네이비 전면 배경
- 좌측 정렬
- 상단에 작은 영문 라벨 `PROJECT BOOK · SCHOOL HEALTH`
- 중앙보다 위쪽에 책 제목
- 제목 아래 가로 구분선
- 구분선 아래 부제
- 하단에 저자명
- 흰색 제목과 청회색 보조 텍스트를 사용한다.

### 2. 차례

- 흰색 배경
- `차례` 제목을 크게 배치
- PART와 Chapter를 텍스트 목록으로 구성
- PART명은 굵게, Chapter명은 그 아래에 배치
- 별도의 장식 이미지 없이 텍스트 위계로 정리한다.

### 3. PART 표지

- 흰색 배경
- 상단 또는 상단 중앙 영역에 작은 `PART n`
- 그 아래에 그린색 PART 제목
- 제목 아래 짧은 그린 가로선
- 페이지 하단에 책의 성격을 설명하는 문구
- Chapter 표지와 달리 밝은 페이지를 사용한다.

### 4. Chapter 표지

- 네이비 전면 배경
- 우측 상단에 매우 큰 Chapter 번호를 낮은 대비로 배치
- 제목 위에 짧은 선과 `PART · CHAPTER` 라벨
- 중앙 영역에 흰색 Chapter 제목
- 제목 아래 긴 구분선
- 하단에 책 제목을 작은 글자로 배치
- Chapter 제목은 내용 단위에 맞춰 여러 줄로 나뉜다.

### 5. Chapter 첫 본문

- Chapter 표지 다음 페이지에 프로젝트 진행 카드를 먼저 배치한다.
- 진행 카드 아래에서 현장 장면 또는 이전 Chapter 결과를 연결하는 본문이 시작된다.
- 별도의 큰 삽화 없이 글의 장면과 카드가 진입 역할을 한다.

### 6. 일반 본문

- 흰색 바탕의 한 단 구성
- 짧은 문단 사이에 충분한 수직 여백
- 본문 중간의 소제목은 16pt 굵은 글자로 구분
- 인용 문장은 본문보다 안쪽으로 들여쓴다.
- 표, 목록, 상자는 본문 폭 안에서 배치한다.
- 페이지 하단에 페이지 번호와 작은 하단 정보가 반복된다.

### 7. PART 마무리

- `PART n을 마치며` 제목으로 시작한다.
- 해당 PART에서 만든 결과와 다음 단계로 이어지는 내용을 본문 형식으로 정리한다.
- 일부 마무리 페이지에는 `오늘 바뀐 것`과 `다음 Chapter Preview`가 함께 배치된다.
- 별도 전면 배경을 쓰지 않고 일반 본문 페이지의 시각 체계를 유지한다.

### 8. 에필로그

- 일반 본문 페이지의 흰색 배경과 본문 글꼴을 유지한다.
- 새로운 시각 체계를 추가하지 않는다.
- 앞에서 사용한 문장 중심의 페이지 구성을 끝까지 유지한다.

## 반복 컴포넌트

### 프로젝트 진행 카드

- 연한 민트 배경
- 회색 외곽선
- 좌측 그린 강조선
- 카드 상단에 `프로젝트 진행 카드`
- DAY, PART, Chapter 진행 수, 예상 시간, 난이도, 오늘 만드는 것을 줄 단위로 제시
- Chapter 진입부에서 작업 범위와 결과물을 한눈에 보여준다.

### 소제목

- 16pt의 네이비 계열 굵은 글자
- 본문 문단과 위아래 여백으로 분리
- 번호를 붙이는 경우와 문장형 제목을 사용하는 경우가 함께 확인된다.

### CASE

- 제목 앞에 `CASE ·` 라벨을 붙인다.
- 실제 보건 업무의 한 장면이나 특정 자료를 예로 든다.
- 본문 흐름 안에서 제목과 여백으로 구분하며, 별도의 전면 페이지는 사용하지 않는다.

### Workflow

- 제목 앞에 `Workflow ·` 라벨을 붙인다.
- 단계, 화살표, 표 또는 짧은 설명을 사용해 업무 순서를 보여준다.
- CASE 뒤 또는 실습 앞에 놓이는 구성이 반복된다.

### 실습

- `실습 nn — 제목` 형식을 사용한다.
- Chapter마다 번호가 이어지며 최종 `실습 22`까지 확인된다.
- 설명, 단계, 작성 표 또는 확인 목록으로 구성된다.
- 실습 제목·도입·첫 표가 한 페이지에서 시작하도록 조판된 사례가 제작 기록에서 확인된다.

### 표

- 머리글은 네이비 바탕과 흰색 글자
- 본문 셀은 흰색 바탕과 회색 구분선
- 열 수와 행 수는 내용에 따라 달라진다.
- 장식용 표가 아니라 Before/After, 역할, 행동, 기준, 입력·출력 등 구조 비교에 사용된다.
- 작은 표 안에서는 7.5–7.8pt 글자가 사용된다.

### 기본 안내 상자

- 흰색 또는 매우 연한 청회색 배경
- 얇은 회색 테두리
- 필요할 때 좌측 그린 강조선
- 수행 확인, 다음 Chapter Preview와 보조 설명에 사용된다.

### 주의·해석 상자

- 연한 크림 배경
- 좌측 골드 강조선
- 본문에서 따로 확인해야 하는 해석이나 주의 문장을 담는다.
- 강한 경고 아이콘 대신 색과 선으로 구분한다.

### 오늘 바뀐 것

- 네이비 머리글과 흰색 본문 셀을 조합한 Before/After 표
- Chapter 결과를 이전 상태와 이후 상태로 비교한다.
- Chapter 후반부에 반복 배치된다.

### 다음 Chapter Preview

- 연한 청회색 상자
- 다음 Chapter에서 이어질 질문이나 작업을 짧게 제시한다.
- 현재 Chapter의 결과가 다음 Chapter의 입력이 된다는 연결을 시각적으로 보여준다.

### QR 영역

- QR 이미지와 `QR · 『보건교사를 위한 AI 업무 자동화』 자료실` 제목을 함께 배치한다.
- QR 오른쪽 또는 주변에 자료실 접속 후 해당 PART 자료를 찾는 설명을 둔다.
- QR은 흰색 바탕과 충분한 주변 여백을 유지한다.
- QR 페이지는 Chapter 후반의 실습 결과·수행 확인 영역과 함께 나타난다.

## 이미지 사용

### 확인된 규칙

- 최종 PDF에서 확인된 래스터 이미지는 QR 21개와 Chapter 5의 화면 이미지 1개다.
- QR 외의 화면 이미지는 책 전반에 반복되지 않는다.
- 사진, 인물 일러스트, 장식용 AI 이미지가 반복 사용되지 않는다.
- 따라서 이 출간본의 중심 시각 언어는 이미지가 아니라 텍스트, 표, 선, 카드와 여백이다.
- Chapter 5의 화면 이미지는 실패한 첫 화면을 설명하는 실제 사례 자료로 사용된다.

## 구분선과 테두리

- 가는 회색 선이 표, 카드와 상자의 기본 경계에 사용된다.
- 그린 세로선은 진행·구조·일반 강조를 표시한다.
- 골드 세로선은 주의·해석 성격의 강조를 표시한다.
- 표지와 Chapter 표지에서는 청회색 가로선이 제목 영역을 나눈다.
- 굵은 테두리나 둥근 입체 버튼 형태는 확인되지 않는다.

## 여백과 밀도

- 제목 페이지는 넓은 빈 공간을 남긴다.
- 본문은 한 페이지를 문장으로 가득 채우기보다 문단, 소제목, 표와 상자 사이의 간격을 유지한다.
- 표와 카드 내부에는 본문보다 작은 글자를 사용하되 테두리와 패딩으로 구획한다.
- 페이지 끝에 다음 구성 요소를 억지로 붙이지 않고 다음 페이지로 넘긴 사례가 확인된다.
- 제작 기록에서는 고아 문단, 빈 페이지, 표 분리, 제목 줄바꿈과 실습 시작 위치를 별도로 재검수했다.

## 일관성과 예외

### 반복해서 확인되는 일관성

- A5 세로 판형
- 네이비·그린 중심 색상
- Noto Sans KR 계열
- 한 단 본문
- Chapter별 독립 표지
- 민트 진행 카드
- 네이비 표 머리글
- 회색 선과 넓은 여백
- Chapter 후반의 결과 확인·QR·Before/After·Preview 구성

### 확인된 예외

- PART 표지는 Chapter 표지와 달리 흰색 바탕이다.
- Chapter 5에는 다른 Chapter에서 반복되지 않는 실제 화면 이미지가 포함된다.
- PART 마지막 Chapter에서는 일반 Preview 대신 PART 마무리로 연결되거나, PART 마무리와 Preview가 함께 나타나는 구성이 있다.
- 에필로그는 프로젝트 진행 카드·실습·QR 없이 일반 본문 체계를 사용한다.
- 페이지 내용의 길이에 따라 표와 상자의 높이, 한 페이지에 들어가는 컴포넌트 수는 달라진다.

## 이 문서에서 확정하지 않는 항목

다음 내용은 최종 출간본만으로 제작 의도를 단정할 수 없으므로 이 문서의 규칙으로 확정하지 않는다.

- 특정 디자인 도구나 편집 프로그램을 선택한 이유
- 글꼴을 선택한 미적 의도
- 색상이 상징하는 의미에 대한 제작자의 설명
- 새로운 책에서도 반드시 같은 색과 수치를 사용해야 한다는 요구
