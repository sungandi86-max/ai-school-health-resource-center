# 보건교육 영상 대본 가이드

버전: v1.0
업데이트: 2026-07-14

## 준비 정보

- 학습 대상의 학교급
- 영상 길이
- 학습 목표
- 반드시 포함할 공식 지침
- 학생이 영상 후 수행할 활동

## 권장 대본 구조

1. 오늘 배울 내용
2. 학생이 이미 알고 있을 만한 상황
3. 핵심 개념 두세 가지
4. 가상 사례
5. 학생이 확인할 행동
6. 핵심 내용 요약

## AI 요청 예시

```text
아래 정보를 바탕으로 보건교육 영상 대본 초안을 작성해주세요.

학교급: {{학교급}}
영상 길이: {{영상길이}}
학습 목표: {{학습목표}}
핵심 내용: {{핵심내용}}
영상 후 활동: {{활동}}

학생 개인 사례를 만들지 말고 일반화된 가상 상황만 사용해주세요.
공식 지침을 확인할 수 없는 경우 임의로 작성하지 말고 담당자 확인 항목으로 남겨주세요.
```

## 최종 검토

- 학생 개인의 건강정보가 포함되지 않았는지 확인합니다.
- 날짜와 공식 지침은 최신 자료로 확인합니다.
- 학교급에 맞는 어휘와 영상 길이인지 확인합니다.
