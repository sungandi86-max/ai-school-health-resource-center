# 『보건교사를 위한 AI 업무 자동화』 부록 자료 패키지

이 패키지는 전자책 Resource Hub에 연결할 수 있도록 구성했습니다.

## 폴더

- `prompts/` : Chapter 01~22 복사용 프롬프트 Markdown
- `practice/` : Chapter 01~22 개별 실습자료 Excel
- `projects/` : PART 1~8 프로젝트 파일 + 전체 프로젝트 파일
- `resource-manifest.json` : 사이트 데이터 연결용 파일 목록

## 권장 사이트 동선

QR 스캔 → 자료실 → PART 선택 → Chapter 선택 → 프롬프트 복사 또는 실습자료 다운로드

## 개인정보 원칙

실제 학생 이름, 학번, 반 번호, 연락처, 질병명, 검사 결과, 상담 기록은 입력하지 않습니다. 더미 데이터로 먼저 시험하고, 결과는 보건교사가 최종 확인합니다.
