# Chapter 02. 나를 자꾸 같은 자리로 돌려보내는 일을 적었습니다

## 반복 장면 기록지 프롬프트

아래는 최근 일주일 동안 기록한 보건실 업무 장면입니다. 반복되는 행동만 찾아주세요.

[업무 기록]

다음 기준으로 표로 정리합니다.
- 반복된 장면
- 반복 횟수
- 다시 찾거나 되풀이한 것
- 줄이면 안 되는 일 / 구조를 살펴볼 일

사람의 판단이 필요한 일은 자동화 후보로 제안하지 마세요.

## 사용 전 확인

- 실제 학생 이름, 학번, 질병명, 검사 결과, 상담 내용을 입력하지 않습니다.
- 더미 데이터로 먼저 시험합니다.
- 결과는 보건교사가 최종 확인합니다.
