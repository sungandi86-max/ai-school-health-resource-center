# Chapter 16. 처음으로 업무가 끝까지 이어졌습니다

## 첫 Workflow 완성본 프롬프트

아래 Workflow를 실제 업무 한 건으로 처음부터 끝까지 점검해주세요.

[Workflow]
[업무 한 건]

확인 항목:
- 시작점
- 단계별 확인 사실
- 다음 행동
- 사람이 판단한 지점
- 중간에 멈췄을 때 재시작 위치
- 완료 기준

같은 정보를 다시 찾거나 적은 순간도 별도로 표시해주세요.

---

## 사용 전 확인

- 실제 학생 이름, 학번, 질병명, 검사 결과, 상담 내용을 입력하지 않습니다.
- 더미 데이터로 먼저 시험합니다.
- 결과는 보건교사가 최종 확인합니다.
