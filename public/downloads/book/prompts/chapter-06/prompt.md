# Chapter 06. 하나를 고칠수록 다른 곳이 어긋났습니다

## 사용자 흐름 실패 지도 프롬프트

아래 온라인 업무 화면을 처음 사용하는 사람의 순서로 점검해주세요.

[메뉴·카드·링크 설명]

사용자 행동 순서에 따라 표로 정리합니다.
- 사용자가 한 행동
- 기대한 결과
- 실제 결과
- 문제 유형(사용자 언어/업무 식별/화면·기기/계정·권한/공개 범위)
- 다음에 확인할 곳

아직 공통 원인을 확정하지 마세요.

---

## 사용 전 확인

- 실제 학생 이름, 학번, 질병명, 검사 결과, 상담 내용을 입력하지 않습니다.
- 더미 데이터로 먼저 시험합니다.
- 결과는 보건교사가 최종 확인합니다.
