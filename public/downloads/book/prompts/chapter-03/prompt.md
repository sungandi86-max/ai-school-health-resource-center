# Chapter 03. 이 일은 끝까지 사람이 해야 합니다

## 사람·AI·정보 경계표 프롬프트

다음 보건업무를 사람·AI·정보 경계로 나눠주세요.

[업무]

출력 항목:
1. 사람이 직접 판단·결정할 것
2. AI가 보조할 수 있는 반복 정리
3. 최종 확인할 사람
4. 입력·공개 가능한 정보
5. 내부에서만 관리할 정보
6. AI 입력·공개 금지 정보

의학적 판단, 응급 대응, 상담, 개인정보, 공개 여부, 법적 책임은 사람에게 남겨주세요.

---

## 사용 전 확인

- 실제 학생 이름, 학번, 질병명, 검사 결과, 상담 내용을 입력하지 않습니다.
- 더미 데이터로 먼저 시험합니다.
- 결과는 보건교사가 최종 확인합니다.
