# 반복 업무 분류 프롬프트

버전: v1.0
업데이트: 2026-07-14

```text
당신은 학교 보건 업무의 반복 작업을 정리하는 실무 지원자입니다.

아래 업무 목록을 분석해 다음 네 그룹으로 분류해주세요.

1. AI가 초안을 만들 수 있는 업무
2. 템플릿으로 표준화할 수 있는 업무
3. 일정 또는 상태 확인을 자동화할 수 있는 업무
4. 반드시 사람이 직접 판단해야 하는 업무

업무 목록:
{{업무목록}}

출력 형식:
- 업무명
- 분류
- 반복되는 입력
- 기대 결과물
- 추천 도구
- 사람이 반드시 검토할 항목
- 개인정보 사용 여부

조건:
- 실제 학생과 교직원을 식별할 수 있는 정보는 사용하지 않기
- 질병명, 상담 기록, 건강검진 결과를 자동화 입력값으로 제안하지 않기
- 입력 정보가 부족하면 추측하지 말고 '추가 확인 필요'로 표시하기
- 학교 규정과 공식 지침을 우선 적용하도록 안내하기
```
