# ChatGPT 보건 업무 실전 프롬프트

버전: v1.0
업데이트: 2026-07-14

이 자료의 모든 입력 예시는 가상 정보입니다. 실제 학생 및 교직원의 이름, 연락처,
건강정보, 상담 기록을 입력하지 마세요.

## 1. 교직원 안내문 초안

```text
당신은 학교 보건 업무 문서를 정리하는 실무 지원자입니다.

아래 정보를 바탕으로 교직원에게 보낼 안내문 초안을 작성해주세요.

학교급: {{학교급}}
안내 대상: {{대상}}
업무명: {{업무명}}
제출 자료: {{제출자료}}
제출 기한: {{제출기한}}
제출 방법: {{제출방법}}
문의처: {{문의처}}

작성 조건:
- 제목과 핵심 내용을 먼저 제시하기
- 제출 기한과 방법을 구분해서 표시하기
- 입력되지 않은 정보는 임의로 만들지 않기
- 공손하고 간결한 학교 업무 문체 사용하기
- 최종 발송 전 담당자가 날짜와 사실관계를 확인하도록 안내하기
```

## 2. 가정통신문 구조 점검

```text
다음 가정통신문 초안을 검토해주세요.

검토 기준:
1. 보호자가 해야 할 행동이 분명한지
2. 일정과 제출 방법이 서로 모순되지 않는지
3. 어려운 행정 용어를 쉬운 표현으로 바꿀 수 있는지
4. 학생 또는 교직원을 식별할 수 있는 정보가 포함되었는지
5. 학교 규정이나 공식 지침을 다시 확인해야 할 문장이 있는지

결과는 '확인 항목 / 발견 내용 / 수정 제안' 표로 작성해주세요.
확인할 수 없는 사실은 추측하지 말고 '담당자 확인 필요'라고 표시해주세요.
```

## 3. 일정 목록 정리

```text
아래 가상 업무 일정을 주차별 실행 목록으로 정리해주세요.

기간: {{운영기간}}
업무 목록: {{업무목록}}
담당 구분: {{담당구분}}

출력 형식:
- 주차
- 준비할 일
- 안내할 일
- 확인할 일
- 완료 기준

개인의 건강정보를 포함하지 말고 업무 단위 정보만 사용해주세요.
```

## 4. AI 결과 최종 검토

```text
다음 AI 작성 결과를 학교에서 사용하기 전에 검토해주세요.

검토 항목:
- 날짜, 기관명, 법령명처럼 사실 확인이 필요한 표현
- 실제 개인정보 또는 민감정보
- 학교 규정과 충돌할 가능성이 있는 내용
- 대상이 오해할 수 있는 모호한 문장
- 담당자가 최종 결정해야 하는 부분

원문을 바로 확정하지 말고, 수정 제안과 담당자 확인 항목을 분리해서 작성해주세요.
```

## 사용 전 확인

- 모든 변수를 가상 또는 업무 단위 정보로 바꿉니다.
- AI가 작성한 날짜, 법령, 기관명은 공식 자료로 다시 확인합니다.
- 최종 발송 또는 게시 전 사람이 문장을 검토합니다.
