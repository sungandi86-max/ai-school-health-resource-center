# AI Workflow 설계서

버전: v1.0
업데이트: 2026-07-14

실제 학생 및 교직원의 개인정보를 입력하지 말고, 업무 단위의 가상 정보로 먼저
설계하세요.

## 1. 업무 목표

- 반복되는 업무:
- 줄이고 싶은 수작업:
- 최종 결과물:
- 반드시 사람이 결정할 부분:

## 2. 입력 자료

- 업무명:
- 대상 범주:
- 일정:
- 필요한 참고 자료:
- 사용하면 안 되는 개인정보:

## 3. 처리 단계

1. 입력 자료에서 개인정보를 제거하거나 범주화합니다.
2. AI로 초안, 분류, 요약 중 필요한 작업을 수행합니다.
3. 템플릿에 결과를 옮깁니다.
4. 날짜, 기관명, 규정 관련 내용을 사람이 확인합니다.
5. 최종 결과를 학교의 공식 경로로 전달합니다.

## 4. 도구와 역할

| 도구 | 사용하는 이유 | 사람이 확인할 항목 |
| --- | --- | --- |
| ChatGPT | 안내문 초안과 업무 구조 정리 | 사실관계와 문체 |
| Google Sheets | 업무 단위 진행 현황 정리 | 접근 권한과 입력 범위 |
| Codex | 코드와 프로젝트 구조 점검 | 실행 결과와 보안 설정 |

## 5. 완료 기준

- 결과물의 목적과 대상이 분명합니다.
- 개인정보가 포함되지 않았습니다.
- 학교 규정과 최신 지침을 확인했습니다.
- 최종 결과를 사람이 검토했습니다.
