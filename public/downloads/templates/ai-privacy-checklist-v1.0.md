# AI 사용 전 개인정보 체크리스트

버전: v1.0
업데이트: 2026-07-14

## 입력 전

- [ ] 학생 이름, 학번, 반 번호가 포함되지 않았습니다.
- [ ] 교직원 이름, 연락처, 개인 제출 상태가 포함되지 않았습니다.
- [ ] 질병명, 건강검진 결과, 상담 기록이 포함되지 않았습니다.
- [ ] 학교 내부에서만 사용하는 비공개 문서 원문을 그대로 붙여넣지 않았습니다.
- [ ] 필요한 경우 정보를 학급, 학년, 업무 단위로 범주화했습니다.

## AI 사용 중

- [ ] AI가 입력되지 않은 날짜, 기관명, 법령을 임의로 만들지 않도록 요청했습니다.
- [ ] 결과에 실제 개인정보가 새로 포함되지 않았는지 확인했습니다.
- [ ] 민감한 판단을 AI에게 맡기지 않았습니다.

## 최종 공유 전

- [ ] 학교 규정과 최신 공식 지침을 다시 확인했습니다.
- [ ] 일정, 제출 기한, 문의처를 담당자가 검토했습니다.
- [ ] 최종 문장을 사람이 읽고 수정했습니다.
- [ ] 결과물에 가상 예시가 실제 사례처럼 남아 있지 않은지 확인했습니다.

이 체크리스트는 학교의 공식 개인정보 처리 절차를 대체하지 않습니다.
