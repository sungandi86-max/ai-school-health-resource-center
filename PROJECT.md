# AI 보건교사 자료실 프로젝트 개요

## 프로젝트 개요

AI 보건교사 자료실(AI School Health Resource Center)은 『보건교사를 위한 AI 업무 자동화』 전자책과 연계되는 보건교사 전용 AI 실무 자료 플랫폼입니다. 보건교사가 학교 현장에서 바로 참고할 수 있는 프롬프트, 템플릿, 코드, 체크리스트, Workflow, 프로젝트 사례를 한곳에서 찾고 활용하도록 돕습니다.

## 목표 사용자

- 초중고 및 특수학교 보건교사
- 보건 업무 자동화를 처음 시도하는 교사
- 전자책 독자와 강의 수강자
- 학교 보건 업무 자료를 직접 개선하려는 AI 실무 사용자

## 해결하려는 문제

- 보건 업무별 AI 활용 자료가 흩어져 있어 다시 찾기 어렵습니다.
- 프롬프트만 제공하면 실제 업무 절차와 연결하기 어렵습니다.
- 전자책 출간 이후 변경되는 예시와 도구 사용법을 지속적으로 갱신해야 합니다.
- 개인정보 보호 원칙을 지키면서 연습할 수 있는 안전한 가상 예시가 필요합니다.

## 핵심 가치

- 빠른 탐색: 업무, 도구, 난이도, 전자책 장 기준으로 자료를 찾습니다.
- 바로 활용: 프롬프트 복사와 템플릿 다운로드를 중심으로 설계합니다.
- Workflow 연결: 단일 자료가 실제 업무 흐름 안에서 어떻게 쓰이는지 보여줍니다.
- 안전한 예시: 실제 학생 및 교직원 개인정보 없이 가상 데이터만 사용합니다.
- 지속 업데이트: 전자책과 강의 이후에도 최신 자료를 유지합니다.

## 주요 메뉴

- 홈
- 자료 찾기
- Prompt Library
- Templates
- Workflows
- Projects
- 전자책 연계 자료
- Updates
- 이용 안내

## MVP 범위

- 모바일 우선 홈 구조
- 통합 자료 목록과 검색
- 자료 상세 페이지
- 프롬프트 복사
- 템플릿 다운로드 링크
- Workflow, Resource, Project 사례 간 연결
- 전자책 장별 자료 연결
- 업데이트 내역
- 이용 안내와 개인정보 보호 안내

## 제외 범위

- 로그인과 사용자 계정
- 데이터베이스와 Supabase 연동
- 즐겨찾기, 개인 자료함, 검색 통계
- 관리자 CMS
- 실제 학생, 보호자, 교직원 개인정보를 포함한 예시
- 자동으로 학교 규정이나 최신 공식 지침을 대체하는 기능

## 개인정보 보호 원칙

- 모든 샘플 데이터는 가상 데이터로 작성합니다.
- 학생 이름, 학번, 반 번호, 연락처, 질병명, 건강검진 결과, 상담 기록을 예시에 포함하지 않습니다.
- AI 도구에 개인정보나 민감정보를 입력하지 말라는 안내를 자료 상세와 이용 안내에 반복 배치합니다.
- 보건 관련 결과물은 학교 규정과 최신 공식 지침을 확인한 뒤 사용하도록 안내합니다.
- 개인정보가 필요한 실제 업무는 익명화, 요약, 범주화된 정보만 사용하도록 유도합니다.
