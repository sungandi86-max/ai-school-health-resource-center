# AI 보건교사 자료실

> 보건교사의 AI 업무, 여기서 시작하세요.
> 프롬프트를 넘어 Workflow로.

**AI 보건교사 자료실(AI School Health Resource Center)**은 보건교사가 학교 현장에서 AI를 실제 업무에 적용할 수 있도록 프롬프트, 템플릿, 코드, 체크리스트, 실습 자료와 Workflow를 제공하는 실무 자료 플랫폼입니다.

## 프로젝트 목표

이 프로젝트는 『보건교사를 위한 AI 업무 자동화』 전자책과 연계되는 자료실에서 시작합니다.

책에는 QR코드를 제공하고, 최신 프롬프트와 추가 자료는 이 사이트에서 지속적으로 업데이트합니다.

단순한 파일 다운로드 페이지가 아니라 다음 목표를 가진 플랫폼으로 개발합니다.

* 업무에 필요한 자료를 빠르게 검색
* 프롬프트를 한 번에 복사
* 템플릿과 실습 파일 다운로드
* 업무별 AI 활용 Workflow 확인
* 실제 프로젝트 사례와 연결
* 전자책 출간 이후에도 자료 지속 업데이트

```text

→ 관련 자료 탐색

```

## 콘텐츠 관리

초기 버전에서는 Markdown 또는 MDX 기반으로 자료를 관리합니다.

```text
content/
├─ resources/
├─ prompts/
├─ workflows/
├─ projects/
└─ updates/
```

각 자료는 제목, 설명, 업무 분류, 도구, 난이도, 버전, 업데이트일, 관련 전자책 장 등의 정보를 포함합니다.

## 개인정보 보호 원칙

이 자료실에서 제공하는 예시와 실습 자료에는 실제 학생 및 교직원의 개인정보를 포함하지 않습니다.

사용자는 AI 도구에 다음 정보를 입력하지 않아야 합니다.

* 학생 이름
* 학번 및 반 번호
* 연락처
* 질병명
* 건강검진 결과
* 상담 기록
* 기타 민감정보

AI가 생성한 결과는 학교 규정과 최신 공식 지침을 확인한 후 사용해야 합니다.
