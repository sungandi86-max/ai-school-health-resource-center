# AI 보건교사 자료실 Roadmap

## MVP 1.0

### 범위

- 홈
- 자료 찾기
- 자료 목록과 상세 페이지
- Prompt Library
- Templates
- Workflows
- Projects
- 전자책 연계 자료
- Updates
- 이용 안내
- 모바일 우선 반응형 UI
- 프롬프트 복사와 템플릿 다운로드 링크

### 완료 조건

- 주요 메뉴가 모두 접근 가능합니다.
- 가상 샘플 자료로 목록, 상세, 관련 자료 연결을 확인할 수 있습니다.
- 검색과 필터가 클라이언트 데이터 기준으로 동작합니다.
- Project Center에서 프로젝트 사례, 관련 Workflow, 관련 Resource 연결을 확인할 수 있습니다.
- 개인정보 보호 안내가 이용 안내와 자료 상세에 표시됩니다.
- `npm run lint`, `npm run build`, `git diff --check`가 통과합니다.

## Version 1.1

### 범위

- 변수 입력형 프롬프트
- 관련 자료 추천
- 최근 본 자료
- 인기 자료 영역
- 복사와 다운로드 이벤트 기록을 위한 클라이언트 구조 준비

### 완료 조건

- 사용자가 변수 입력 후 프롬프트를 복사할 수 있습니다.
- 자료 상세에서 관련 자료 추천이 표시됩니다.
- 개인 저장 없이 브라우저 수준의 최근 본 자료가 동작합니다.

## Version 1.2

### 범위

- MDX 또는 Markdown 기반 콘텐츠 관리 고도화
- 전자책 장별 랜딩 페이지 개선
- 업데이트 로그 자동 정렬
- 콘텐츠 작성 가이드 정비

### 완료 조건

- 콘텐츠 frontmatter 검증 기준이 문서화됩니다.
- 전자책 장별 자료 연결이 누락 없이 표시됩니다.
- 새 자료 추가 절차가 반복 가능하게 정리됩니다.

## Version 2.0

### 범위

- 사용자 계정
- 즐겨찾기
- 개인 자료함
- 학교급별 맞춤 자료
- 자료 업데이트 알림
- 관리자 CMS
- Supabase 등 데이터베이스 연동 검토

### 완료 조건

- 로그인 전후 정보구조가 분리됩니다.
- 개인정보와 민감정보를 저장하지 않는 정책이 명확히 구현됩니다.
- 데이터베이스 보안 정책과 접근 권한이 검증됩니다.
- 관리자 운영 흐름이 문서화됩니다.
